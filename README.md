# CursorConverter

A program to convert mouse cursors between formats.

## Supported:

Convert **from** and **to**:
<div>

- `.ani`
- `.ico`
- `.xcg`
- `.cur`
- `.CursorFX`
- `.CurXPTheme`
- `hyprcursor`
- `jpg`
- ani, ico, xcg, cur, png, jpg, hyprcursor, cursorfx, curxptheme
